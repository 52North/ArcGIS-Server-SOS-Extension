This package contains Java representations of the database model types which describe the SOS service itself and are not covered by types of the packages {@link org.n52.om} or {@link org.n52.gml}. These include the {@link org.n52.sos.dataTypes.ServiceDescription}, {@link org.n52.sos.dataTypes.ContactDescription}, {@link org.n52.sos.dataTypes.ObservationOffering}, and also {@link org.n52.sos.dataTypes.Procedure} as well as {@link org.n52.sos.dataTypes.ObservedProperty}.
